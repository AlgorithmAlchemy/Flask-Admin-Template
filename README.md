# Flask-Admin-Template
Этот проект использует Flask-Admin для создания административной панели. Вы можете кастомизировать шаблон и стили для вашего проекта, чтобы создать уникальный интерфейс для работы с данными. Включает поддержку различных моделей данных через SQLAlchemy и простое управление записями.
